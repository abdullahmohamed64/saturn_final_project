# saturn

Saturn is a Flutter-based social media application that allows users to share their creativity and connect with others through posts, comments, chats, and favorite collections.
Designed with a modern interface and smooth performance, Saturn provides an engaging platform for art, ideas, and everyday moments.

✨ Features

🖼️ Posts:
Users can create, edit, and delete posts with text and images to share their thoughts or artwork.

💬 Comments:
Engage with other users through comments on posts in real time.

💌 Chats:
Built-in private messaging system for direct user-to-user communication.

⭐ Favorites Page:
Save your favorite posts to view them anytime.

👤 User Profiles:
Each user has a personalized profile page showing their posts, info, and interactions.
